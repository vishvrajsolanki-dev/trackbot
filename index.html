Future Development
Ideas Board

Features, upgrades, and research directions tracked for TrackBot 2 and beyond.

Planned — TrackBot 2
Scissor Fork Lift Mechanism
Expert-fabricated aluminium scissor lift. True vertical linear motion. NEMA 17 stepper. 500g payload capacity.
Planned — TrackBot 2
Autonomous Charging Dock
Robot navigates to charging station when cell voltage alarm triggers below 3.6V/cell. No manual intervention.
Planned — TrackBot 2
Stepper-Based Rack and Pinion Claw
Contingency mechanism — NEMA17 rack-and-pinion replaces servo claw if servo torque insufficient for payload. Budget allocated in Z.
Planned — Research
Multi-Robot Fleet via MQTT
MQTT broker manages multiple TrackBot units. Central task scheduler with shared occupancy grid for collision avoidance.
Planned — Research
ArUco Marker Navigation
Replace RFID floor tags with ArUco visual markers. OV7670 reads markers for position correction. No physical floor hardware.
Planned — Research
PID Auto-Tuning (Ziegler-Nichols)
Relay feedback auto-tuning for all 4 wheel PID controllers. Eliminates manual tuning session. Better performance across surfaces.
